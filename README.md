# React based blog 

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

To install this project run

```bash
pnpm run setup
```

This will install the required libraries for front and backend.


First, run the development server:

```bash
pnpm run dev
```

- The front end will run on [localhost:3000](http://localhost:3000)
- The backend end will run on [localhost:5000](http://localhost:5000)
